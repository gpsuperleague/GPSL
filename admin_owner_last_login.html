This is website sign-in only — when the owner last logged into the GPSL site (Supabase Auth). Most recent at the top; longest time since login at the bottom. No emails are shown.

Discord “last login” to the GPSL server is not available from Discord. Their API does not give bots a last-seen / last-login time for server members (only join date, which is not shown here). To track Discord presence we would need a always-on Discord bot that records when members appear online — say if you want that built.
