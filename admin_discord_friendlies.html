Discord Friendlies

Owners post a result in Discord. When the opponent posts the matching result, GPSL pays ₿5,000 each (first 10 / GPSL month, ₿500,000 season cap).

JUB 2 - 2 BEN
ROS 2 - JUB 3

Either format works (en-dashes from Discord are fine).

Auto-detect (hands-free)

Once enabled, Postgres polls Discord every 2 minutes — no Poll now needed. Use the same service_role key as Discord News auto-post (Supabase → Project Settings → API → service_role).

First run supabase/sql/patches/discord_friendlies_cron.sql in the SQL editor (schedules the cron job; copies News key if already saved).
