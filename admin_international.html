International & World Cup

Setup

Refresh selectable nations — imports missing GPDB nationalities, rebuilds the player pool cache, then marks a nation active only if it can form a 23-man squad (≥24 players, ≥2 GKs) and support a GPSL club (same rating-band rules as nation select). Thin nations are deactivated (kept in the DB; assigned nations stay active).

Recompute seed ranks — orders active nations by pool strength (weighted rating bands from nation player pool). Seed 1 = strongest. Used for balanced qualifying pots later — does not change owner draft order.

Run supabase/sql/patches/international_refresh_selectable_and_seed_ranks.sql in Supabase once if these buttons error with a missing function.

Owner rankings

Nation draft order uses rolling points from the last four seasons, computed automatically when a season is archived (Season & calendar admin). Run competition_owner_ranking.sql once, then use recompute for backfill.
