International & World Cup

Setup

Run these one step at a time (each can take up to ~1–2 min). Do not combine them in one SQL query — Supabase times out long combined jobs.

1. Import GPDB labels — add missing nationalities from the player database.
2. Refresh pool cache — recount players by rating band (slowest step).
3. Apply selectable — activate nations with ≥24 GPDB players and ≥2 GKs (23-man squad bar). Club-depth bands on the pool page are informational only.
4. Recompute seed ranks — order active nations by average rating of top 100 GPDB players (seed 1 = strongest for qualifying pots).

First deploy: run supabase/sql/patches/international_refresh_selectable_and_seed_ranks.sql in Supabase. If a button times out, run the matching SELECT … alone in the SQL Editor.

Owner rankings

Nation draft order uses rolling points from the last four seasons, computed automatically when a season is archived (Season & calendar admin). Run competition_owner_ranking.sql once, then use recompute for backfill.
