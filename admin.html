Transfer Management

Wage % of market value

Standard calculated wage = market_value × % by the club's current competition division (SuperLeague vs Championship A/B). Required once in Supabase: run supabase/sql/player_wage_settings.sql (creates admin_update_wage_settings + wage columns). If save fails, run that file then retry.

Special Auctions

Create a lowest-unique or snap auction, then set it as the active special auction owners see on the dashboard.

Draft auctions are hidden until you select one and click Set as active / scheduled.

Owner Administration

When you add an owner, they automatically receive a password‑setup email. The temporary password is only required by Supabase and is never used.

Link existing login to club

For test accounts or users who already signed up: looks up their user id by email and sets Clubs.owner_id. Run admin_assign_club_owner.sql once in Supabase first.
