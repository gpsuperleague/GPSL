Match scheduling (league)

Owner guide: Learning GPSL → Match scheduling. GPSL month = Fri 19:00 UK → Fri 19:00 UK. SQL patches under supabase/sql/patches/match_scheduling_*.sql.

Page / RPCFeatures
fixtures.htmlLeague & cup list — Schedule, catch-up badge, response deadline hints
fixture_schedule.htmlMutual slots, propose/accept, check-in (10 min), Match Day link, voluntary/emergency drop, replay reset, catch-up reset, mutual override
inbox.htmlProposals, agreed times, fines, replay/catch-up notices — accept/counter inline
owner_details.htmlWeekly availability grid, timezone, holidays
admin_fines.htmlManual fines & compensation; scheduling tariffs seeded here

Rules (automated — league only)

  • Arrange — from fixture draw / pre-season; home proposes first; mutual 30-min slots
  • Arrangement fines at month lock — Match Management ₿10m (no proposal), Late arrangement ₿5m (first proposal in last 24h before play month)
  • Response deadlines — tracked in-month; Missed response ₿2.5m at play-month lock if still overdue
  • Check-in — both miss → no fine, replay reset; one misses → recorded, no instant forfeit if match later played normally; month lock → 3–0 + ₿5m if still unplayed
  • Catch-up — play month closed unplayed → flag + reschedule in later active month
  • Cron — gpsl-transfer-engine-5min → calendar month tick (fines throttled; deploy free_tier_cron_throttle.sql on Free/Pro Nano)

Background systems

  • Transfer engine — listing expiry, reserve sales, seller review, draft settlement
  • Player contracts — 3-year default, renew/expire, expiring wage bids, season rollover
  • Squad rules — HG ≥8, U21 ≥5, max 28, overflow release/fine
  • Finance ledger — all ₿ movements via RPCs and transfer completion
  • Stadium attendance v2 — prestige rank, monthly fill drift, gate cap 100%
  • Match scheduling — availability, propose/accept, check-in, replay & catch-up reset, month-lock fines (arrangement, response, no-show); see Match scheduling
  • Inbox — owner notifications
  • Dashboard registry — pin/reorder pages as tiles
