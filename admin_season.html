Season management

GPSL season calendar

One UK week (Fri 19:00 → Fri 19:00) = one GPSL month. Set the anchor before Start season. Run competition_real_world_calendar.sql and admin_season_lifecycle.sql in Supabase first.

End GPSL month early

Locks the live GPSL month now and runs the same automatic jobs as the Friday cron. Optionally opens the next GPSL month immediately and pulls all remaining months forward (each keeps its 7-day play window; season end moves earlier). Re-run competition_admin_end_gpsl_month.sql after updates.

Between GPSL months

No GPSL month is live right now. This usually happens after ending a month without checking “Also unlock next month now”. Open the next month to resume fixtures and scheduling. Run competition_admin_calendar_gap_recovery.sql after updates.

Rebuild GPSL Sport edition

Re-runs the rich in-season generator for a locked month (TOTM, top scorers, Match of the Month). Use this if August still looks bare-bones after SQL patches — applying SQL alone does not update an existing edition row. Run gpsl_sport_inseason_rich_edition.sql first.
