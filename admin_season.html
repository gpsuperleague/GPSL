Season management

GPSL season calendar

One real UK week (Fri 19:00 → Fri 19:00) = one GPSL month for every month, including pre-season. Set season start = when June begins. Then: June (week 1) → July (week 2) → August league start (week 3) → … → Playoffs. Run patches/calendar_june_july_weeks.sql in Supabase first.

Testing: End Month Early (lock live month now / open next month) is under Admin → Testing.

GPSL Sport

To create or rebuild a Sport edition after ending a month early, use Republish GPSL Sport under Season Management.
