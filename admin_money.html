Money management

League prize money

Paid automatically when all 38 league matches in a division are played (or use Pay league prizes manually). Run competition_league_prizes.sql once.

Cup prize money

Round fee paid to both clubs on confirm. Set stages on Cup fixtures (run competition_cup_prizes_fix.sql).

Government subsidies

HG bands are non-cumulative for status (highest tier shown) but payout is per band: Quota ≤5, Flying 6–8, Pride 9+ only. Youth uses banded u21 counts. BnB: squad players at or below max rating. Paid at EOS when all 3 divisions have 38/38 league matches (run government_subsidies.sql).

Wage %

Annual squad wage = market value × division % (run player_wage_settings.sql).

Tax %

League tax on transfers / income is handled via central bank ledger rules (central_bank_phase1.sql). A dedicated tax % admin control is not wired yet — set rates in SQL when introduced.

Stadium costs

Expansion cost per seat tiers (by current capacity). Owners: seats × tier rate. Day-7 cancel penalty goes to Rapid Build Co.

Backfill gate receipts

Run competition_phase5_finances.sql. Settles gates for played fixtures missing ledger rows.
