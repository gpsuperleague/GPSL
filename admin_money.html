Money management

League prize money

Paid automatically when all 38 league matches in a division are played (or use Pay league prizes manually). Run competition_league_prizes.sql once.

Cup prize money

Round fee paid to both clubs on confirm. Set stages on Cup fixtures (run competition_cup_prizes_fix.sql).

Season challenges

Owners see targets on Season challenges (Owners + League nav). Admin config: Admin → Season challenges. Run competition_challenges.sql, then Seed default targets.

TV revenue

Selects big league fixtures per GPSL month per division (top-8 clashes, promotion/relegation battles, dry spells, etc.). Both clubs paid on confirm. Run competition_tv_revenue.sql after government_subsidies.sql.

Government subsidies

HG bands are non-cumulative for status (highest tier shown) but payout is per band: Quota ≤5, Flying 6–8, Pride 9+ only. Youth uses banded u21 counts. Weak squad bonus: 14+ players rated ≤72 → flat ₿10M. Paid at EOS when all 3 divisions have 38/38 league matches (run government_subsidies.sql).

Wage bill & taxes

Season charges: squad wages (sum of contract wages), 34+ rating fee, star maintenance (designated ★ players only — set on Squad). Emergency tax = % on cash above threshold. Run competition_wages_taxes.sql and patches/club_squad_designations.sql.

Wage %

Annual squad wage = market value × division % (run player_wage_settings.sql).

Fines & compensation

Instant gov_fine_compensation ledger lines from the Excel tariff list. Admin → Fines & compensation (run competition_fines.sql).

Tax %

League tax on transfers / income is handled via central bank ledger rules (central_bank_phase1.sql). A dedicated tax % admin control is not wired yet — set rates in SQL when introduced.

Stadium attendance & club prestige

Club rank (not owner rank) from rolling league + cup points sets stature. Display fill drifts monthly toward the season target; gate revenue is capped at 100% (display can exceed 100% as cushion). Floor 75%. Two on-target seasons ≈ full gate. Run competition_club_stadium_attendance.sql then stadium_attendance_v2.sql, then Recompute club rankings.
