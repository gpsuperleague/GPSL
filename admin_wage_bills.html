Close season finances

Runs end-of-season money for the current season, in order:

  1. Wage bills — player wages, manager salary (weekly × 52), 34+ fee, star tax
  2. Debt interest — bank rate (default 5%) on overdrawn balances
  3. FFP — flat ₿10M fine if balance ≤ −₿100M (after interest)
  4. Balance interest — 0.5% credit on positive balances only
  5. Finance archive refresh — updates Season accounts snapshot (so Archive-before-this is safe)

Each line is skipped if already posted for that club/season. Ensure these patches are applied in Supabase: close_finances_eos_interest_ffp.sql, manager_season_salary_wage_bills.sql, and if debt/FFP inserts fail a type check, re-run season1_eos_interest_ffp_backfill.sql ( widens entry types ) or competition_finance_ledger_entry_types_repair.sql.
