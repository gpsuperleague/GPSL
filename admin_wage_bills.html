Close season finances

Runs end-of-season money for the current season, in order:

  1. Wage bills — player wages, manager salary (weekly × 52), 34+ fee, star tax
  2. Debt interest — bank rate (default 5%) on overdrawn balances
  3. FFP — flat ₿10M fine if balance ≤ −₿100M (after interest)
  4. Balance interest — 0.5% credit on positive balances

Each line is skipped if already posted for that club/season. Run patches/close_finances_eos_interest_ffp.sql in Supabase first. Rates/threshold live on gpsl_bank_account (eos_balance_interest_pct, eos_debt_interest_pct or policy_interest_rate_pct, eos_ffp_debt_threshold, eos_ffp_flat_fine).
