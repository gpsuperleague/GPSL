Discord News Feed

Sky-style #gpsl-news

Confirmed results, transfers, listings, and auctions are queued automatically. Use Push news to Discord to drain the queue (or set up a Database Webhook for auto-post).

One-time setup: run SQL patch supabase/sql/patches/gpsl_discord_sky_feed.sql, set secret DISCORD_WEBHOOK_URL, deploy supabase functions deploy discord-sky-feed.
