Season challenges

Defaults

Each club earns the per-challenge prize when a target is met. The first club to complete all challenges in a window (e.g. all 5 Start targets) automatically wins the big prize pack below — cash, medical tokens, transfer fee discounts, and red-card appeal cards. That club is paid instantly, items go to Club prizes, and every owner gets an inbox announcement (plus Discord if the news feed is live). Start = June–December. Mid = January–May.

Big prize pack (first to finish all challenges)

Set this before clubs finish the set. Separate Start / Mid pots. Medical tokens: 2,4,6,8,10 matches (Medical room). Fee discounts: % off what the buyer pays (seller still gets full fee; bank tops up). Appeal cards: red-card overturn requests (admin review). Draft tokens: sign any uncontracted player at market value (Club prizes); if squad is full (28), release a player at MV first. Save packs, then use Recheck all clubs if someone already completed all targets.
