League prize money

Paid automatically when all 38 league matches in a division are played (or use Pay league prizes manually). Run competition_league_prizes.sql once. New seasons auto-copy the previous season’s table (patches/prize_config_persist_across_seasons.sql).

Division copy fills the grid for the division you’re editing. Copy & save also writes it. Copy season copies all divisions from another season into the selected season.
